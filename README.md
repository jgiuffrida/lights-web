# README #

### What is this repository for? ###
Angular Client for trout, party sourced playlists and stuff [buzzwords]

### How do I get set up? ###

#### Quick Start ####
```
npm install
bower install
gulp serve
Browser -> http://localhost:3000
```
#### Testing ####
##### Unit Testing #####
Single Run
```
gulp test
```
Watch/Auto Run
```
gulp test:auto
```

##### e2e Testing #####
Dev tests
```
gulp protractor
```
Dist tests
```
gulp protractor:dist
```
